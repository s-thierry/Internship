🧾APPLICATION GUIDE


Please fill out the form below to register a new student. Follow these steps:
1. Student Name
Enter the student’s First Name and Last Name correctly.

2. Gender
Select the student’s gender from the dropdown menu.

3. Date of Birth
Enter the student’s birth date in the format: dd/mm/yyyy.

4. Class
Write the class or grade the student will be enrolled in (e.g., Grade 5, Senior 2).

5. Parent/Guardian Information
Provide the full name of the parent or guardian responsible for the student.

6. Contact Number
Enter a valid phone number so the school can communicate easily.

7. Address
Fill in the student’s home address clearly (e.g., city, district, street).

8. Review and Submit
Check all details carefully, then click the Register / Submit button.

⚠️Important Notes


1. Make sure all information is accurate.
2. Use a phone number that is always reachable.
3. Double-check spelling of names before submitting.
